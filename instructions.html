How to use...

Opening

Afte opening the webpage application you will see an empty page with various icons on the top

These icons represent various functionality of the application. Clicking on each one of them performs a function.

Main page

Color Pallete

Color Pallete For filtering content by color

Clicking on each color renders the tasks corresponding to that color on the screen.

Clicking on the same color icon twice renders the webpage to the normal view.

NOTE: If there are no tickets/tasks corresponding to that particular color the main page will remain blank.

Plus-Cross icons To add and delete tickets

The plus icon is used to add new tickets to the application.

Selecting and clicking on the tickets on the main page deletes the corresponding ticket/task. To disable this functionality click on the Cross icon again.

Lock-Unlock iconsto enable editing and freezing edit functionality

The lock and unlock icons provide the functionality of freaaing and defreezing the tickets/tasks

When the lock icon is activated, the tickets already present on the main screen cannot be edited while when the unlock icon is selected this can be done by clicking and consequently editing the desired ticket

Adding ticket/task area

On clicking the plus icon, the 'Adding ticket area is popped up.

Click on the color you wnat the task/ticket to be color coded with and type your task in the text area and press enter to produce the ticket/task.


Additional features

Changing color of tickets: Each color coded ticket can have its color code changed by clicking on the header color of the ticket multiple times until the desired color is obtained.

Each ticket also carries a code for unique identification
